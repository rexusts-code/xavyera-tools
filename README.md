# Xavyera Code CLI 🚀

Xavyera Code is a powerful, terminal-native agentic AI coding assistant. Built to mirror the capabilities of Claude Code but with more flexibility and a custom premium UI.

## Features
- **Multi-Provider**: Use Gemini, Claude, OpenAI, Groq, or local models via Ollama.
- **Agentic Loop**: Full access to file system, shell, and web search.
- **Elite Mode**: Specialized `/code` mode for hacking tools and fullstack development.
- **Quota System**: Tiered access (Silver, Gold, Master).
- **Premium UI**: ASCII art and interactive terminal dashboard.

## Installation
You can install Xavyera Code directly using:
```bash
curl -sSL https://raw.githubusercontent.com/rexusts-code/xavyera-tools/main/install.sh | bash
```

## Commands
- `/help`: Show all commands.
- `/code`: Enter Elite Architect Mode.
- `/plan`: Enter Strategic Planning Mode.
- `/status`: Check AI provider and quota.
- `/exit`: Quit.
